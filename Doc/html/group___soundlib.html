◆ STX_sndtimerdiv

enum STX_sndtimerdiv

Enumeration des valeurs possibles pour definir la valeur du diviseur de fréquence du timer D du MFP.

Valeurs énumérées
DIV4 

fréquence timer D de 614400 Hz

DIV10 

fréquence timer D de 245760 Hz

DIV16 

fréquence timer D de 153600 Hz

DIV50 

fréquence timer D de 49152 Hz

DIV64 

fréquence timer D de 38400 Hz

DIV100 

fréquence timer D de 24576 Hz

DIV200 

fréquence timer D de 12288 Hz

◆ STX_dmasampleendframe()

void STX_dmasampleendframe ( )

Fonction appeleé à la fin de chaque lecture d'echantillons DMA si STX_setdmasampleinterrupt() aà été appelée auparavent. .

Note
Cette fonction doit être impérativement implémementée dans le code C, dans le cas contraire une erreur de linkage interviendra.
