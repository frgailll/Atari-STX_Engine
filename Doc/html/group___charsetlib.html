◆ STX_printstring()

void STX_printstring ( unsigned short  carx,
unsigned short  cary,
unsigned char  color,
STX_displaymode  mode,
STX_displaysens  sens,
char *  text 
)

Affiche une chaine de caractère en utilisant les charsets.

Paramètres
carx: Coordonnée X en caractère comprise entre 0 et 39
cary: Coordonnée Y en caractère comprise entre 0 et 24
color: Couleur d'affichage de 0 à 15
mode: Trois mode d'affichage disponible (remplacement, transparence, effacement)
sens: affichage horizontalement (de la gauche vers la droite) ou verticalement (de bas en haut)
*text: pointeur sur une chaine de caractère à 0 terminale
Voir également
STX_displaymode, STX_displaysens
