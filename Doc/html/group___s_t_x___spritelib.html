◆ STX_movespriteex()

short STX_movespriteex ( unsigned short  spriteid,
long int  xincr,
long int  yincr 
)

Permet de definir le deplacement du sprite en X et Y avec des valeurs à virgules.

Paramètres
spriteid: id du sprite concerné
xincr: Valeur du deplacement en X, 16 bits poids fort = unité, 16 bits poids faible = aprés la virgule
yincr: Valeur du deplacement en Y, 16 bits poids fort = unité, 16 bits poids faible = aprés la virgule
Renvoie
0 si pas d'erreur, -1 en cas d'erreur, 1 si sprite disable, 3 si clipping
Note
Pour effectuer un déplacement vers la droite en X de 1,25 pixel il faudra écrire 0x00010019 pour xincr
Pour effectuer un déplacement vers la gauche en X de 1,25 pixel il faudra écrire 0xFFFFFFE7 pour xincr

◆ STX_setanimation()

short STX_setanimation ( unsigned short  spriteid,
unsigned short *  motiflist,
unsigned char  vitesse,
bool  animloop 
)

Defini l'animation d'un sprite par rapport à une liste de motifs contenus dans un tableau. La dernière valeur du tableau doit être 0xFFFF pour marquer la fin de l'animation.

Paramètres
spriteid: id du sprite concerné
motiflist: pointeur sur un tableau contenant les numéros de motifs à afficher successivement, la derniére valeur du tableau doit être 0xFFFF
vitesse: vitesse d'affichage, corresponds au nombre de tour de la boucle infini pour changment de motif
animloop: si true, l'animation sera affichée en boucle, si false, l'animation sera effectuée une seul fois
Renvoie
0 si pas d'erreur, -1 en cas d'erreur ou de sprite non créé au préalable
